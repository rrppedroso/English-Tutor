Get your free Groq API Key 100% free

Groq is genuinely free — no credit card, no billing. It uses the Llama 3 model and is extremely fast.

1
Access console.groq.com/keys and create a free account (can use Google login).
2
Click Create API Key, give it any name, and copy the key (gsk_...).
3
Paste it below and click Start. The key is saved only in your browser.
Your key stays in localStorage — never sent anywhere except Groq's API.
